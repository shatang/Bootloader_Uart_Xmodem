/********************************************************************************
* 函数名 : IAP_Uart_Xmodem
* 功  能 : IAP功能，使用串口，采用Xmodem进行升级
* 说  明 : 基于Stm32F407ZGT6
* 入  参 : none
* 返  回 : none
* 设  计 : Shatang                    时  间 : 2020.09.17
* 修  改 : none                       时  间 : none
********************************************************************************/
1. 该IAP升级，通过超级终端进行串口传输文件，Xmodem协议
2. 文件名、类型不定(与Xmodem协议有关)
3. 该Bootloader进行升级需要重新上电启动
4. 上电0.8s检测是否上位机是否一直发送空格，且满足15个；0.8s后检测不到(或升级完毕)进入App程序
5. 升级过程在超级终端有提示
6. 进行CRC校验；不带加密示例
7. Bootloader程序总大小为6KB
